## ✨ Overview

**CampusConnect** is designed to make campus life smarter and more connected.  
It helps students discover events, join communities, share updates, and stay informed — all in one place.

---

## 🚀 Features

- 📰 **Campus Feed** – Real-time updates and announcements
- 🎉 **Events Hub** – Discover and manage campus events
- 👥 **Clubs & Communities** – Join student groups and interact
- 📌 **Opportunities Board** – Internships, hackathons, workshops, and more
- 🔔 **Notifications** – Never miss important updates
- 🔍 **Search & Filters** – Quickly find relevant posts/events/groups

## ⚙️ Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/onkarjadhav5598/CampusConnect.git

# Go to project folder
cd CampusConnect

# Install dependencies
npm install
```

### Run Locally

```bash
npm run dev
```

App will run at: `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
